# About
Starter Admin template for Laravel, Vue.js, and Bootstrap 4. Inspired by Laravel Nova.

* Includes:
    * ~User Management~
    * ~Permissions~
    * ~Profile~
    * RESTful API (All Components)
    * Bootstrap 4
    * Font Awesome 5.2
    * Vue.js Components
    * Vue Router
