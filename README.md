# About
Starter Admin template for Laravel, Vue.js, and Bootstrap 4. Inspired by Laravel Nova.

* Includes:
    * ~User Management~
    * ~Permissions~
    * ~Profile~
    * RESTful API (with all components)
    * Bootstrap 4
    * Font Awesome
    * Vue.js Components
